# 피부 타입 다중 선택 기능 업데이트

## 📋 변경 내용

### 1. UI 변경
- **이모지 아이콘 제거**: 💧, ✨, 🌗, ⚠️, 😊 → **체크박스**로 교체
- **다중 선택 안내**: 서브 헤더에 "(복수 선택 가능)" 추가
- **체크박스 위치**: 각 피부 타입 카드 왼쪽에 배치

### 2. 기능 변경

#### SkinTypeActivity
- **단일 선택** → **다중 선택**
- 선택한 피부 타입들을 `Set`으로 관리
- 콤마(`, `)로 연결하여 저장: "건성, 지성"
- 체크박스 토글 방식으로 선택/해제
- 1개 이상 선택 시 완료 버튼 활성화

#### ProfileFragment
- **단일 선택 다이얼로그** → **다중 선택 다이얼로그**
- `setMultiChoiceItems()` 사용
- 현재 선택된 피부 타입들을 체크 상태로 표시
- 선택 해제 가능

### 3. 데이터 저장 형식

**이전:**
```
"건성"
```

**현재:**
```
"건성, 지성"
"복합성, 민감성"
"건성, 복합성, 민감성"
```

## 🔄 동작 방식

### 1. 선택 플로우
```
카드 클릭
   ↓
체크박스 토글
   ↓
selectedSkinTypes Set 업데이트
   ↓
완료 버튼 활성화 상태 체크
   ↓
선택된 타입들을 ", "로 연결
   ↓
SharedPreferences에 저장
```

### 2. 표시 형식
- **프로필 화면**: "건성, 복합성"
- **API 요청**: "건성, 복합성" (그대로 전달)
- **다이얼로그**: 개별 체크박스로 표시
