📝 기능 명세서: 사용자 피부 타입 설정 (Onboarding)
1. 개요 (Overview)
앱을 처음 실행하는 사용자에게 **피부 타입(건성, 지성, 복합성 등)**을 선택받아, 향후 OCR 분석 결과에서 개인화된 맞춤 정보를 제공하기 위함입니다. 사용자가 자신의 피부 타입을 모를 경우를 대비해 '나중에 하기' 기능을 제공하며, 설정된 값은 언제든 프로필 탭에서 수정 가능합니다.

2. 사용자 플로우 (User Flow)
A. 앱 최초 실행 시

Splash Screen 종료.

앱 내부 저장소(SharedPreferences)를 확인하여 피부 타입 설정 여부 체크.

설정 안 됨: SkinTypeActivity (피부 타입 선택 화면)로 이동.

설정 됨 (또는 '나중에 하기' 선택 이력 있음): MainActivity로 바로 이동.

B. 피부 타입 선택 화면 (SkinTypeActivity)

사용자가 피부 타입 버튼(CardView/Button) 중 하나를 선택.

[완료] 버튼 활성화 -> 클릭 시 저장 후 메인으로 이동.

또는 우측 상단 [나중에 하기] 텍스트 클릭 -> 저장 없이(또는 UNKNOWN 상태로 저장) 메인으로 이동.

C. 수정 하기 (프로필 탭)

하단 네비게이션의 [프로필] 탭 진입.

"내 피부 타입: OOO" 영역 클릭.

피부 타입 선택 다이얼로그 또는 SkinTypeActivity 재진입하여 수정.
